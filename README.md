图像指纹技术对图片进行去重，核心目标是将一张图片转换成一个固定长度的、紧凑的二进制字符串（哈希值），使得视觉上相似的图片产生的哈希值也相似（即汉明距离小），而视觉上差异大的图片产生的哈希值差异大（即汉明距离大）
- aHash (Average Hash - 平均哈希)
- dHash (Difference Hash - 差异哈希)
- pHash (Perceptual Hash - 感知哈希)
